BZOJ1005 明明的烦恼

传送门-BZOJ

传送门-Codevs

Description

自从明明学了树的结构,就对奇怪的树产生了兴趣……给出标号为1到N的点,以及某些点最终的度数,允许在任意两点间连线,可产生多少棵度数满足要求的树?

Input

第一行为N(0 < N < = 1000),接下来N行,第i+1行给出第i个节点的度数Di,如果对度数不要求,则输入-1

Output

一个整数,表示不同的满足要求的树的个数,无解输出0

Sample Input

3
1
-1
-1

Sample Output

2

HINT

两棵树分别为1-2-3;1-3-2

Prufer序列

把一棵树进行以下操作：
1.找到编号最小的叶节点，删除这个节点，然后与这个叶节点相连的点计入序列
2.反复进行1，直到这棵树只剩下两个节点时，退出

比如说这个图(来自度受百科)
最小叶节点为2，删除2，将3计入序列
最小叶节点为4，删除4，将5计入序列
最小叶节点为5，删除5，将1计入序列
最小叶节点为1，删除1，将3计入序列
图中只剩下两个节点，退出
于是得到这棵树的Prufer序列为{3,5,1,3}

以上来自他的博客
求一个树的Prufer序列，可以用迭代删点的方法做，到只剩两个点
所以得到一个长度为n-2的序列

Prufer序列的性质

  1. 唯一性，一个Prufer序列对应一个树（如果相同，每个节点的子节点和出入度相同，那么树也相同）
  2. 一个点若度数为d，则一定在Prufer序列中出现了d-1次

已知一个树的节点在Prufer序列中出现了几次，问有几种不同的Prufer序列
