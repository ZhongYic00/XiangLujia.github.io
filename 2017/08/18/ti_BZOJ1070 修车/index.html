BZOJ1077 修车

传送门

Description

同一时刻有N位车主带着他们的爱车来到了汽车维修中心。维修中心共有M位技术人员，不同的技术人员对不同的车进行维修所用的时间是不同的。现在需要安排这M位技术人员所维修的车及顺序，使得顾客平均等待的时间最小。 说明：顾客的等待时间是指从他把车送至维修中心到维修完毕所用的时间。

Input

第一行有两个m,n，表示技术人员数与顾客数。 接下来n行，每行m个整数。第i+1行第j个数表示第j位技术人
员维修第i辆车需要用的时间T。

Output

最小平均等待时间，答案精确到小数点后2位。

Sample Input

2 2
3 2
1 4

Sample Output

1.50

HINT

数据范围: (2<=M<=9,1<=N<=60), (1<=T<=1000)

Source

题解

一开始看的时候觉得这个明明是个大水题，后来一想又不对，如果只是车连人的话怎么控制每个人每个时刻修一个车，等的时候的时间也不好算
还是看了题解
因为数据范围很小，那么就可以把m个人分成n个点（一个人最多修n个车），第i个表示这个车是倒数第i个修的，那么倒数第i个修对时间的贡献是mp[i][j]*k(因为别人都开走了，越后修贡献越少)
